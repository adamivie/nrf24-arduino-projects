# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial project structure
- Wireless chat application with bidirectional communication
- Range testing system with packet loss analysis
- Comprehensive diagnostic tools
- Detailed wiring and troubleshooting documentation
- Cross-platform dependency installation scripts
- GitHub Actions for automated testing

### Features
- **Wireless Chat System**
  - Real-time bidirectional text communication
  - Clean Serial Monitor interface
  - Visual feedback for message status
  - Character echo and proper formatting

- **Range Testing Tools**
  - Continuous packet transmission with counters
  - Real-time packet loss calculation
  - Signal strength estimation
  - LED status indicators

- **Diagnostic Utilities**
  - Hardware connection testing
  - SPI communication verification
  - Multiple pin configuration support
  - Low-level register access testing

### Documentation
- Complete wiring guide with diagrams
- Comprehensive troubleshooting guide
- Contributing guidelines
- Installation scripts for all platforms
- API documentation

### Hardware Support
- Arduino Uno (primary)
- Arduino Nano (tested)
- Arduino Leonardo (tested)
- Standard NRF24L01 modules
- NRF24L01+PA+LNA modules (compatible)
